# ✨ PureTouchUs - Professional Cleaning Solutions

PureTouchUs is a premium, modern web application designed for a professional cleaning service. It features a stunning, high-performance interface with a focus on user experience, interactive elements, and a seamless booking flow.

## 🌟 Key Features

- **🚀 Performance Optimized**: Built with Vite and React for lightning-fast load times.
- **📱 Fully Responsive**: A mobile-first design that looks great on all devices.
- **📅 Interactive Booking**: A user-friendly booking section with date selection and service options.
- **✨ Visual Gallery**: Interactive before/after showcase to demonstrate quality work.
- **💬 Testimonials**: Dynamic carousel for customer feedback and reviews.
- **❓ Comprehensive FAQ**: Collapsible FAQ section to answer common customer queries.
- **🎨 Premium Aesthetics**: Custom design system using Tailwind CSS and shadcn/ui.
- **🎭 Smooth Animations**: Powered by Framer Motion for a fluid, high-end feel.

## 🛠️ Tech Stack

- **Core**: [React 18](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) (Radix UI)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Form Handling**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Data Fetching**: [TanStack Query](https://tanstack.com/query/latest)
- **Routing**: [React Router](https://reactrouter.com/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [Bun](https://bun.sh/)

### Installation

1.  **Clone the repository**:
    ```sh
    git clone <repository-url>
    cd PureTouchUs
    ```

2.  **Install dependencies**:
    ```sh
    npm install
    # or
    bun install
    ```

3.  **Start the development server**:
    ```sh
    npm run dev
    # or
    bun run dev
    ```

### Production Build

To create a production-ready bundle:
```sh
npm run build
```

## 📂 Project Structure

- `src/components`: Reusable UI components and page sections.
- `src/pages`: Main page components (e.g., Index.tsx).
- `src/hooks`: Custom React hooks for shared logic.
- `src/lib`: Utility functions and configurations.
- `public`: Static assets like images and icons.
